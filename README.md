# Agata Jeżow - Psychotherapy Website

A responsive single-page website for a Gestalt psychotherapist based in Szczecin.

## Features
- **Responsive Design:** Fully compatible with mobile, tablet, and desktop devices.
- **Sections:** - Bio and professional background.
  - Areas of clinical focus (individual and couples therapy).
  - Pricing and session duration details.
  - Contact section with integrated Google Maps and "ZnanyLekarz" booking link.

## Tech Stack
- **HTML5:** Semantic structure.
- **Tailwind CSS:** Styling via CDN.
- **Google Fonts:** Using *Playfair Display* for headings and *Inter* for body text.

## Assets
- **Logo & Images:** Sourced from the local `/img` directory in the repository.

## Setup
To view the website:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. An active internet connection is required to load CSS styles and fonts from CDNs.
